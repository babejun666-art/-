# 👴👵 银发知音：孤寡老人智能陪伴助手

## 📖 项目简介

随着老龄化社会的到来，许多孤寡老人面临着孤独、缺乏倾诉对象以及数字鸿沟等问题。本 AI 助手的核心初衷是“用科技传递温度”，为长辈们提供一个耐心、温暖、全天候的智能倾听者和对话伴侣，帮助他们缓解孤独感，解答生活中的日常疑问。

## ✨ 核心愿景
* **情感陪伴**：用温和、易懂的语言与老人进行日常交流，倾听他们的故事。
* **跨越鸿沟**：提供简单直接的交互方式，帮助中老年人也能享受 AI 带来的便利。
* **生活助手**：耐心解答健康养生常识、防骗指南及日常繁琐小问题（注：本应用不提供专业医疗诊断）。
